Install XAMPP

  • Install from http://apachefriends.org
  • Just agree with everything.
  • Default settings are quite OK.
  • You can leave out 'BitNami'. We don't need it.

Configure xdebug

  1. Stop apache, if it is running
  2. Open C:\xampp\php\php.ini
  3. Go to the bottom of the file
  4. Uncomment zend_extension = "C:\xampp\php\ext\php_xdebug.dll" 

[XDebug]
zend_extension = "C:\xampp\php\ext\php_xdebug.dll"
;xdebug.profiler_append = 0
;xdebug.profiler_enable = 1
;xdebug.profiler_enable_trigger = 0
;xdebug.profiler_output_dir = "C:\xampp\tmp"
;xdebug.profiler_output_name = "cachegrind.out.%t-%s"
;xdebug.remote_enable = 0
;xdebug.remote_handler = "dbgp"
;xdebug.remote_host = "127.0.0.1"
;xdebug.trace_output_dir = "C:\xampp\tmp"
XDebug hebben we niet perse nodig. Maar het helpt heel erg met debuggen.

Add PHP to PATH

  1. Start, right click on 'Computer' -> Properties
  2. Column on the left, 'Advanced System Settings'
  3. In tab 'Advanced', click the button 'Environment Variables...'
  4. Under 'System Variables', find 'Path'
  5. Add ;C:\xampp\mysql\bin;C:\xampp\php; to the end.

This is where Mac users should do something else for a while.

On Windows 8: Control panel -> System. Then continue with step 2 (Advanced System Settings)

We're not done yet!

The frontpage is working. But we still have some things to configure.

Database

  1. Open phpmyadmin
  2. Create a new database, you should name it something like gewisweb_dev

Database - config

In the gewisweb config autoload directory (C:\xampp\htdocs\gewisweb\config\autoload), copy doctrine.local.php.dist to doctrine.local.php and open it.

Database - config

Fill in the username, password and database name

This should be root, no password and gewisweb_dev


<?php
return array(
    'doctrine' => array(
        'connection' => array(
            'orm_default' => array(
                'driverClass' =>'Doctrine\DBAL\Driver\PDOMySql\Driver',
                'params' => array(
                    'host'     => 'localhost',
                    'port'     => '3306',
                    'user'     => 'root',
                    'password' => '',
                    'dbname'   => 'gewisweb_dev',
                )
            )
        )
    )
);

Database - initialization

  1. Open a shell (in GitHub application, right click -> open a shell here)
  2. Execute ./vendor/bin/doctrine-module orm:schema-tool:create
